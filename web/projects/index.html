Mobius

Sidd Vadgama

@siddvee siddv.net

Mobius is a platform for creating wireless Arduino projects with web-based control and zero code. Using standardized, accessible and cost-effective components Mobius allows you to deploy projects easily and with little financial cost. Mobius generates both, the web-based control and the Arduino code, allowing you to create huge portions of projects without writing a single line of code.

All you have to do select a control method (slider, colour picker, number etc) and a template that for what the device does (motors, servos, lights etc). Then Mobius generates the control method and code, along providing you with a wiring diagram. This lowers the barrier of entry for anyone with ideas, but not the technical prowess to pursue them. Mobius is a completely open and expandable system too, allowing you to upload your own templates and add your own control inputs. If you’re a developer it’s easy to mess with the system too due to its modular design.
